通过Xrdp实现Windows远程访问Ubuntu 16.04

2019-03-05
  • Ubuntu
  • Ubuntu

目前网上的大量教程都是需要安装xfac4或者xubuntu桌面系统才能实现远程连接。因为xrdp支持在13.10之后版本就已经不支持的Gnome了和原生Unity桌面，所以网上很多方法都是安装能够被xdrp支持的第三方xfac4或者xubuntu桌面系统间接达到远程控制Ubuntu。
本文提供如何使用Xrdp访问原生Ubuntu桌面。
